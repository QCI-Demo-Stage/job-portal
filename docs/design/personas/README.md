# User Personas — Job Portal MVP

This directory is the **shared design repository** for primary user personas. Personas synthesize stakeholder interviews and scope workshops conducted for the Job Portal MVP (Epic: Requirements & Specification Documentation).

| Persona | File | Primary role |
|---------|------|----------------|
| Employer | [employer-persona.md](./employer-persona.md) | Post jobs, manage applications |
| Candidate | [candidate-persona.md](./candidate-persona.md) | Search, apply, track status |
| Admin | [admin-persona.md](./admin-persona.md) | Moderation, users, reporting |

The **Candidate** persona is also published to the [project wiki](../../wiki/personas/candidate-persona.md) for stakeholder access.
