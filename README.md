# Job Portal API

Express + TypeScript backend scaffold with ESLint, Prettier, and layered folders (`routes`, `controllers`, `services`, `models`, `middleware`).

## Scripts

- `npm run dev` — run with hot reload (`tsx watch`)
- `npm run build` — compile to `dist/`
- `npm start` — run compiled output
- `npm run lint` / `npm run format` — lint and format
